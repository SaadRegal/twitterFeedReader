Run your app

If you're not yet running the static server, you can do so by running the following command on your terminal:

npm start

Twitter API

The proxy server should be already running in the background. If you provided correct credentials for your Twitter API and Twitter Secret Key, you should now be able to use the api as the example below:

http://localhost:7890/1.1/statuses/user_timeline.json?count=30&screen_name=makeschool
